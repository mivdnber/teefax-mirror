To navigate the pages, either press a colour button or type the number in directly.

If there is anything you'd like to see added to Teefax then please email your ideas to peterk.vt80@gmail.com.

You can also create your own pages by using this online editor.

When you have designed a page, bookmark it to save it and then copy the URL and send it to the above address.

If you are more ambitious you can use wxTED, which is a Windows application. It has support for most teletext features.

If your TV has the teletext feature then try the Raspberry Pi project on page 785. It takes these pages and sends them to a TV. The cost is about £50 consisting of £35 for a Raspberry Pi and £15 for software and cables. Again if you would like to try this then drop an email to the address above.
